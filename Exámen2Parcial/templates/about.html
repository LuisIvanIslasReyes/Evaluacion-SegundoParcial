Proceso de Desarrollo

El desarrollo de este sistema siguió las siguientes etapas:

  1. Análisis Exploratorio de Datos (EDA) - Comprensión del dataset de diabetes
  2. Preprocesamiento - Limpieza y normalización de datos
  3. Entrenamiento de Modelos - Evaluación de múltiples algoritmos
  4. Optimización - GridSearchCV para encontrar los mejores hiperparámetros
  5. Validación - Evaluación exhaustiva con múltiples métricas
  6. Despliegue - Implementación de la aplicación web con Flask
